A Frozen Minecraft World

Created by Stephanie Cheng and Nicholas Gadjali

Role: Programmer

This project was inspired by a childhood favorite of many. It features imported terrain from a Minecraft world built by Nicholas and his friends during their teenage years. In this game, the user can navigate a flame-propelled ship throughout the city to explore its beauty. The game features gravity, turbulence, and restitution to create a realistic and immersive experience for the player. This game was programmed using the OpenFrameworks C++ toolkit in Xcode. While this was my first time working with 3-Dimensional elements, I greatly enjoyed the challenges and creative freedom that came with building this game. Camera setup and ship model by Nicholas Gadjali.
